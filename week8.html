5. What's my pick-of-the-week or tip-of-the-week?

I guess I'm going to plug the stuff the project I'm working on is based off of. Intel's SGX is a very cool hardware feature, present in all modern Intel chipsets, that provides some serious security features for untrusted computation. Ryoan is a framework built by Tyler Hunt, Professor Emmett Witchel, and some other students here at UT. It takes advantage of SGX to create super-secure sandboxes within "enclaves" that protect data from not only the operating system, but even from the applications running within the enclaves and processing the data. It's a pretty neat idea, and we might start to see its applicablity to providers of software-as-a-service in the future.
